Розбий наданий текст юридичного документу на семантичні секції:

- FACTS: Фактичні обставини справи
- CLAIMS: Позовні вимоги
- LAW_REFERENCES: Посилання на норми права та законодавство
- COURT_REASONING: Судове обґрунтування та мотивування
- DECISION: Резолютивна частина рішення
- AMOUNTS: Суми, штрафи, пеня, компенсації

Документ для аналізу:
{{user_query}}

Поверни JSON об'єкт з полем "sections" - масивом об'єктів з наступними полями:
- type: тип секції (одне з перерахованих вище)
- text: текст секції
- start_index: позиція початку в оригінальному тексті
- end_index: позиція кінця в оригінальному тексті
- confidence: впевненість в класифікації (0.0-1.0)

Приклад формату відповіді:
{
  "sections": [
    {
      "type": "FACTS",
      "text": "...",
      "start_index": 0,
      "end_index": 500,
      "confidence": 0.9
    }
  ]
}
